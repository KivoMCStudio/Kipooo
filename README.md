# Kipooo
一款普通的原版增强插件

## 使用提示

1. 新建模块

> 你需要实现一个模块接口才可以新建一个模块
> 在org.kivo.kipooo.modules中
> Cli是交互消息模块
> Complete是补全模块
> Essentials是基础模块（所有模块都要实现的接口）
> Message是信息模块
> Sound是声效模块
> Timer是计时器模块

2. 提交

> 你需要在提交记录中写出你的提交信息
> 或者在patch文件中声明

3. 协议

**本插件使用MIT协议开源**

4. 问题与BUG

> 可以在Github上提个issue，按照给予的优先级来处理
